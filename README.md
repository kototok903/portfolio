# Portfolio

Personal programming portfolio built with bun, Astro, Tailwind v4, and shadcn/ui primitives.

## Commands

```bash
bun install
bun run dev
bun run check
bun run build
bun run preview
```

## Environment

The contact form uses Resend through an Astro Action.

```bash
RESEND_API_KEY=...
CONTACT_EMAIL=...
```

`CONTACT_EMAIL` is optional and defaults to the current portfolio inbox.
