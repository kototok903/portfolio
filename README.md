# Portfolio

Personal programming portfolio built with Astro, Bun, Tailwind CSS, and shadcn-style UI primitives.

## Commands

```bash
bun install
bun run dev
bun run check
bun run build
bun run preview
```

## Environment

The contact form uses Resend through an Astro Action.

```bash
RESEND_API_KEY=...
CONTACT_EMAIL=...
```

`CONTACT_EMAIL` is optional and defaults to the current portfolio inbox.
